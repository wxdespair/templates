## python 项目的结构模板



### 0. 介绍

为python项目建立一个较为通用的结构模板。

### 1. 使用方法

#### 1.1 目录结构

```
├─data : 存放项目所用数据文件
├─docs : 存放项目相关文档
├─images : 存放项目产生的图片
├─logs : 存放项目产生的日志文件
├─src : 存放项目使用的代码文件
├─src_tmp : 临时使用的代码文件
├─test : 项目使用的测试目录
└─tools : 存放一些独立工具，比如当前存放了两个工具文件：
	├─print_docstring.py : 打印所有 py 文件中的函数文档
	└─update_readme.py : 更新 README 文件中的最后修改时间，使用正则表达式匹配修改内容，需要满足格式规则
```
